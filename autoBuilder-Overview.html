AutoBuild

NAuto.AutoBuild removes the need for tedious and repetitive data creation when test driving code and also when test data is required for integration testing.

Key Features

  • Automatically populate nearly any object *
    • Including deep object graphs of complex types, lists, dictionaries etc.
    • You can limit how deep into the graph NAuto will go if required
    • For objects without default constructors, you can either pass in specific parameters or let NAuto do it for you... it will pretty much construct anything correctly :)
  • Conventions based solution makes taking fine grained control of data creation very simple
  • Configuration solution allows for simple & sharable custom implementations
  • Easily create data in different languages including:
    • English
    • Chinese
    • Russian
    • Pinyin
    • Italian
    • German
    • Spanish
  • Override anything using "With" methods
  • Override using If-Then logic
  • Persist models as JSON
  • Load models from JSON
  • Follows the Test Builder Pattern

Is it any good?

Yes, probably the best test data builder since sliced bread **
