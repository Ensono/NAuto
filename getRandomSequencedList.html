Get Random Sequenced List

Get random sequenced list is actually just an override of GetRandomList.

Spinning up a random sequenced list is both very simple and also very powerful.

A property must be selected to be the sequenced property and must be of type int.

Generating a random sequenced list. 
var newList = NAuto.GetRandomList<MyModel>(seq => seq.Id)

The example will return a list with the id property sequenced starting from 1 and incrementing by 1

There are a few additional properties that can be used to configure the sequenced list for example:

var newList = NAuto.GetRandomList<MyModel>(
            seq => seq.Id,
            numberOfItems: 10,
            seed: 5,
            increment: 5,
            autoBuilderConfiguration: config,
            language:Language.Chinese))

Each list item is created via NAuto.AutoBuild therefore each list item has access to the feature set available within AutoBuild such as Conventions and Multi-Language support.

Configuration

The GetRandomList method optionally accepts an AutoBuilderConfiguration object.

For more information on AutoBuilderConfiguration, please see the Configuration section.

Generating a random list using customer configuration. 
var newList = NAuto.GetRandomList<MyComplexType>(
              seq => seq.Id,
              autoBuilderConfiguration: myCustomConfig);

Language Support

The GetRandomList method optionally accepts a language parameter.

For more information on languages, please see the Languages section.

Generating a random list in traditional Chinese. 
var newList = NAuto.GetRandomList<MyComplexType>(
              seq => seq.Id,
              language: Language.Chinese);
