If-Then

AutoBuilder accepts If-Then syntax for altering data generation based on user defined conditions.

Shows first name being altered depending on start of email string 
var myModel = NAuto.AutoBuild()
    .Construct()
    .If(x => x.Email.StartsWith("foo"))
    .Then(x => x.FirstName = "Foo")
    .Build();
