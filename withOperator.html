Using the With operator

NAuto allows any object to have its test data auto generated based on conventions. However there will always be the need to override some of the values auto generated with specific values. NAuto provides this through the With operator.

Lets start with an example:

If we had the a very simple object as below

public class User
{
    public string Name { get; set; }

    public int Age { get; set; }

    public string Email { get; set; }

    public string Password { get; set; }
}

We can simply use NAuto to auto generate the model test data with the following command:

var user = NAuto.AutoBuild()
            .Construct()
            .Build();

However say we had the scenario where we wanted to auto generate everything except for in this case the email address and password properties. You can then use the With statement to override with specific values as follows:

var user = NAuto.AutoBuild()
            .Construct()
            .With(x => x.Email = "a@b.com")
            .With(x => x.Password = "pa55word")
            .Build();

The With statement takes a Func to allow you to write flexible lamda expressions
