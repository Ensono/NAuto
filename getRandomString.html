Get Random String

NAuto allows you to easily acquire a random string value to use in your tests. The default overload allows you to specify the number of characters the string should contain.

Generating a random double with a specified minimum and maximum value. 
var randomStringOfTwentyCharacters = NAuto.GetRandomString(20);

In addition to be able to specify the length of the string, NAuto provides a number of additional overloads on the GetRandomString operation to allow you to generate a more precise random string.

These are:

CharacterSetType

Allows you to specify whether the string should contain:
CharacterSetType.Alpha Contains only letters [A-Z]
CharacterSetType.AlphaNumeric Contains letters and numbers [A-Z][0-9]
CharacterSetType.Numeric Contains only numbers [0-9]
CharacterSetType.Anything Contains letters, numbers and special characters

Example usage:

var value = NAuto.GetRandomString(20, CharacterSetType.AlphaNumeric);

Language

Allows you to get a random string in the specified language. The languages that are supported are:
Language.English
Language.Chinese
Language.German
Language.Italian
Language.Pinyin
Language.Russian
Language.Spanish

If a language is not specified, then English is used by default

Example usage:

var value = NAuto.GetRandomString(20, CharacterSetType.AlphaNumeric,
                                      Language.German);

Spaces

Allows you to specify whether any spaces should be included within the random string value. Spaces allow you to specify at what position in the string spaces are included. The available options are as follows:
Spaces.Any
Spaces.Start
Spaces.Middle
Spaces.End
Spaces.StartAndEnd
Spaces.None

If Spaces are not specified then the random string will not contain any spaces

Example usage:

var value = NAuto.GetRandomString(20, CharacterSetType.AlphaNumeric,
                                      Spaces.Middle);

Casing

Casing allows you to specify what case should be applied to the string:
Casing.Any Applies random casing to the string
Casing.Lowered The string generated will be all lower case
Casing.Uppered The string generated will be all upper case
Casing.ProperCase The first letter of each word separated by a space in the string is in upper case

Example usage:

var value = NAuto.GetRandomString(20, CharacterSetType.AlphaNumeric,
                                      Spaces.Middle,
                                      Casing.ProperCase);

Min and Max Length range

GetRandomString provides the option to specify a length range for which the length of the random string will be within.

Example usage:

The following random string will be between 10 and 20 characters in length. 
var value = NAuto.GetRandomString(10, 20, CharacterSetType.AlphaNumeric,
                                          Spaces.Middle,
                                          Casing.ProperCase,
                                          Language.German);
