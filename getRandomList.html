Get Random List

Spinning up a random list is both very simple and also very powerful.

The syntax for spinning up a new random list is as follows:

Generating a random list. 
var newList = NAuto.GetRandomList<MyComplexType>(5);

The example will return a list of MyComplextTypes with 5 items.

Each list item is created via NAuto.AutoBuild therefore each list item has access to the feature set available within AutoBuild such as Conventions and Multi-Language support.

Configuration

The GetRandomList method optionally accepts an AutoBuilderConfiguration object.

For more information on AutoBuilderConfiguration, please see the Configuration section.

Generating a random list using customer configuration. 
var newList = NAuto.GetRandomList<MyComplexType>(
              5,
              autoBuilderConfiguration: myCustomerConfig);

Language Support

The GetRandomList method optionally accepts a language parameter.

For more information on languages, please see the Languages section.

Generating a random list in traditional Chinese. 
var newList = NAuto.GetRandomList<MyComplexType>(
              5,
              language: Language.Chinese);
