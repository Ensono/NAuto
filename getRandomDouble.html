Get Random Double

NAuto allows you to easily acquire a random double value to use in your tests.

The GetRandomDouble operation accepts two parameters allowing you to specify the range in which NAuto should generate a random double by specifying the minimum and maximum allowed values to be returned:

  • Passing Min & Max Integer
Generating a random double with a specified minimum and maximum value. 
var value = NAuto.GetRandomDouble(10.5, 5000.9);
