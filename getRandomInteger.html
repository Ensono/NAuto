Get Random Integer

The GetRandomInteger method returns a random integer, no major surprise there.

There are 2 ways to call this method:

  • Passing Max Integer
  • Passing Min & Max Integer
Generating a random integer with a specified maximum value. 
var newInteger = NAuto.GetRandomInteger(10000);
Generating a random integer with specified minimum & maximum values. 
var newInteger = NAuto.GetRandomInteger(100, 10000);
