Get Random Property Type

This method provides a shortcut for creating common property types, in this release the following types are supported:

  • Email
  • Url
  • Postal Code
  • Telephone Number
Generating a random email address. 
var email = NAuto.GetRandomPropertyType(PropertyType.Email);

Each property type can optionally be passed a language from the list of supported languages, for example this would allow an url to be created using a Russian character set:

Generating a random url with Russian character set. 
var url = NAuto.GetRandomPropertyType(
              PropertyType.Url,
              Language.Russian);
